\msection{Block UBHPPOLY}\label{UBHPPOLY}\hypertarget{UBHPPOLY}{}
\mindex{UBHPPOLY}
\pdfdest name {inselBlock.UBHPPOLY} fit
The HPPOLY block simulates a compression heat pump by using
the polynomial functions that describe the compressor
performance according to EN12900.
